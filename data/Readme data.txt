#####    Datos ordenados .csv   #####
- Si el archivo datos.csv no se ve en esta carpeta ejecuta el codigo src/etl.py 

El archivo **data/datos.csv** incluye información de distintos países obtenida de la API del Banco Mundial. Para cada país se guardan:
- Nombre del país
- Código ISO3
- Región
- Año (1980–2023)
- Esperanza de vida
- PIB per cápita

Esta tabla permite analizar cómo han cambiado estas variables a lo largo del tiempo.
